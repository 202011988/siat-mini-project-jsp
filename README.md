<!-- GETTING STARTED -->
## Getting Started
1.  Clone the repo
   ```sh
   git clone https://github.com/SIAT-Python-Project/KBO-Match-Result-Mail-System.git
   ```
2. Install libraries
  ```sh
pip install selenium
pip install pandas
pip install requests
pip install bs4
pip install schedule
  ```
3. Setting
  ```sh
  cd ./KBO-Match-Result-Mail-System
  mkdir ./IO
  ```

4. Run main.py
  ```sh
python ./src/main.py
  ```
